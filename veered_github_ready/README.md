# Veered

Expo app starter for Veered.

## Run locally

1. Install Node.js if needed.
2. In the project folder run:

```bash
npm install
npx expo start
```

## Put on GitHub

Create a repo named `veered` on GitHub, then run:

```bash
git init
git add .
git commit -m "Initial Veered app"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/veered.git
git push -u origin main
```

## Put online as website

```bash
npx expo export -p web
```

Then deploy the exported web build using a static host such as Netlify or Vercel.
